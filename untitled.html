Data analysis bootcamp is for all of you who have to work with data and work with data sets. This is a set of practical, hands-on training where you will learn how to work with shell, git, bash scripts, R, Rstudio, dplyr and ggplot2. After working through this set of tutorials, you will gain confidence to work with your own data sets. If you feel stuck, please come back to revisit this page and ask questions.
We will cover the following topics:
  1. Basic shell commands
  2. How to use git for version control
  3. How to use OpenRefine for data cleaning and data preprocessing
  4. Working with Rstudio for data analysis using tidydata, dplyr, and ggplot2 for data analysis and graphing
  5. How to use lavaan for structural equation modelling
  6. How to write your research using markdown/latex/overleaf/authorea
  7. How to conduct meta analyses and systematic reviews

Basic Shell commands

Shell is a command line interface where you can issue instructions to your computer in plain text and you can get yoru work done. Depending on the kind of operating system you use, you will need to either install or learn to use a shell already bundled with your OS.
  • For windows OS, use powershell or if you install git for windows you can use the gitbash that gets installed on your machine
  • For LInux and Mac OS, you get bash
You can learn more about shell and practice if you follow the data/software carpentry exercise here:
Software carpentry bash tutorial
If you are use Mac or Linux OS, then bash shell is already built in; if you use Windows, please follow the tutorial written by howtogeek.com to install bash shell on Windows 10. If this will not work for you, then you can also download and install win-bash for windows.
